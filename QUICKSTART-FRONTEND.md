# LikeCodex 前端升级 - 快速启动指南

## 🚀 立即体验新功能

### 1. 启动开发服务器

```bash
cd likecodex/web
npm run dev
```

或从项目根目录：

```bash
cd likecodex
npm run dev:web
```

### 2. 访问应用

打开浏览器访问：`http://localhost:3000`

## 🎯 首次使用体验

### 新用户引导

首次访问时，系统会自动显示3步引导教程：

1. **模式选择** - 了解Ask/Agent/Manual三种模式
2. **输入框** - 学习如何描述任务和使用@提及
3. **快捷键** - 掌握常用快捷操作

💡 **提示**: 可以跳过引导，后续通过右上角的 `?` 按钮重新查看

## 🎨 主要新功能演示

### 💬 1. 中心化输入框（对标Cursor）

**位置**: 聊天面板底部中央

**特点**:
- 更大的输入区域（56px高度）
- 渐变背景光效（悬停时显示）
- 圆角设计更加现代
- 语音输入按钮（右侧麦克风图标）

**使用方法**:
```
直接输入你的任务描述...
例如："创建一个React组件来显示用户列表"
```

### 🔄 2. Agent模式切换器

**位置**: 输入框上方的胶囊式按钮组

**三种模式**:

#### 💬 Ask模式（绿色）
- **用途**: 只读问答，不修改代码
- **场景**: 询问代码逻辑、最佳实践
- **示例**: "这段代码是什么意思？"

#### 🤖 Agent模式（蓝色）
- **默认**
- **用途**: 自动执行，最小化确认
- **场景**: 日常开发任务
- **示例**: "重构这个函数，添加错误处理"

#### ✋ Manual模式（橙色）
- **用途**: 每个步骤都需要确认
- **场景**: 重要修改、生产环境
- **示例**: "删除未使用的导入语句"

**切换方法**: 点击对应的胶囊按钮

### 📊 3. Agent Activity进度条

**位置**: 聊天消息中的工具调用区域

**功能**:
- ✅ 实时显示任务完成百分比
- 🔄 运行中任务显示旋转图标
- ✔️ 已完成任务显示绿色对勾
- 📈 渐变色进度条（蓝→紫）

**示例**:
```
Agent Activity [75%]
├─ ✔️ Read file: src/App.tsx
├─ ✔️ Search code: "useState"
├─ 🔄 Edit file: src/App.tsx (running)
└─  Run command: npm test (pending)
```

### 📝 4. 富文本消息展示

**支持的格式**:

#### 代码块
```typescript
// 自动语法高亮
const greeting = "Hello, LikeCodex!";
console.log(greeting);
```

**特性**:
- 语言标签显示（TypeScript、Python等）
- 右上角"Copy"按钮一键复制
- 优化的滚动体验

#### 文件引用

使用 `@文件名` 语法引用文件：

```
请帮我优化 @[App.tsx](src/components/App.tsx) 的性能
```

**显示效果**:
- 📄 文件图标
- 文件名高亮显示
- 可选行号范围（:10-20）
- 点击可跳转查看

### @ 5. 增强版@提及功能

**触发方式**: 在输入框中输入 `@`

**下拉菜单特性**:
- 📁 **分类图标**: 文件、文件夹、符号、问题、Git等
- 📊 **元数据**: Token估算、关联度评分
- ⌨️ **键盘导航**: ↑↓ 选择，Enter确认，Tab快速选择
- 🔍 **实时搜索**: 输入即搜索，100ms防抖
- 📱 **响应式**: 自适应屏幕位置

**支持的上下文类型**:
- `@file` - 文件引用
- `@folder` - 文件夹
- `@symbol` - 代码符号（函数、类）
- `@issue` - GitHub/GitLab问题
- `@git` - Git提交、分支

### ⌨️ 6. 完整的快捷键系统

**打开方式**: 
- 点击右上角 `?` 图标
- 或按 `Ctrl+K` 打开命令面板后搜索"shortcuts"

**核心快捷键**:

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+K` | 命令面板 | 快速执行命令 |
| `Ctrl+B` | 切换侧边栏 | 显示/隐藏左侧面板 |
| `Ctrl+,` | 设置 | 打开配置面板 |
| `Ctrl+N` | 新会话 | 创建新的对话 |
| `Ctrl+Enter` | 发送消息 | 提交输入内容 |
| `Cmd+I` | Composer | 多文件编辑模式 |
| `Cmd+J` | 终端 | 切换终端面板 |
| `Esc` | 关闭 | 关闭弹窗/选择器 |
| `↑↓` | 导航 | 在@提及列表中移动 |
| `Tab` | 快速选择 | 选中当前@提及项 |
| `?` | 帮助 | 显示此快捷键列表 |

## 🐛 常见问题

### Q: 输入框太小了？

A: 新版输入框已经增大到56px高度，如果还是觉得小，可以在设置中调整。

### Q: @提及不显示文件？

A: 确保后端API正常工作，检查 `/api/ide/context/search` 端点是否返回正确数据。

### Q: 快捷键不起作用？

A: 
1. 确认焦点不在其他输入框中
2. 检查是否有浏览器扩展冲突
3. 尝试刷新页面

### Q: 动画卡顿？

A: 
1. 检查浏览器硬件加速是否开启
2. 在设置中降低动画质量
3. 关闭不必要的浏览器标签页

### Q: 引导教程没出现？

A: 可能之前已经看过，清除localStorage中的 `likecodex-onboarding-seen` 键即可重新显示。

## 💡 进阶技巧

### 技巧1: 组合使用模式和@提及

```
切换到 Agent 模式 → 输入 "@src/components/Button.tsx 添加disabled状态支持"
```

### 技巧2: 使用Plan模式进行复杂任务

```
输入 "/plan" → 描述复杂重构任务 → AI生成详细计划 → 逐步执行
```

### 技巧3: 批量文件操作

```
@file1.ts @file2.ts @file3.ts 统一修改这些文件的import语句
```

### 技巧4: 快速导航

```
Ctrl+K → 输入 "open" → 选择文件 → Enter
```

### 技巧5: 撤销操作

```
在Manual模式下，每个操作前都会请求确认
可以随时拒绝不需要的更改
```
